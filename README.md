# Kinder Morgan (kinder-morgan)
Kinder Morgan is one of the largest energy infrastructure companies in North America, owning and operating pipelines and terminals that transport natural gas, refined petroleum products, crude oil, carbon dioxide, and other commodities. Kinder Morgan publishes pipeline informational postings and capacity data through Electronic Bulletin Board (EBB) portals; no public REST API or developer program has been identified at this time.
